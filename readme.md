# Craft 2 + Mix

## Setup 

1. Add babel config: rename babelrc to .babelrc


2. Add a database 
- Sequel is nice.



3. Install: 

`yarn install` 



4. Configure Settings

- db.php (specify database and domain/proxy)
- webpack.mix.js (specify browser sync proxy)
- general.php (specify domain/proxy)



5. Go Time
`yarn watch`
